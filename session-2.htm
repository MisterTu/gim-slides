HTML: Texte strukturieren (Beispiele)

  • <h1> </h1> – Überschriften (h1 bis h6)
  • <p> </p> – Absätze
  • <em> </em> – Betonung (Emphasis)
  • <strong> </strong> – Starke Betonung
					body {
						font-family : sans-serif;
					}
					h1 {
						color : green;
					}
					p {
						padding : 20px;
						background-color : black;
						color : white; 
					}

CSS: Texte formatieren

  • font-family – Schriftfamilie
  • font-weight – Fette
  • font-style – Kursive
  • font-variant – Kapitälchen
  • font-size – Schriftgröße
  • line-height – Zeilenabstand
  • text-decoration – Unterstreichung u.ä.
font-weight
  • Numerisch: 100, 200, 300 bis 900
  • Alternativ: normal, bold, bolder und thinner
  • Abhängig von der Verfügbarkeit eines entsprechenden Schriftschnitts
font-size / line-height
  • Numerisch (absolute Einheiten): px (Pixel), pt (Punkt)
  • Numerisch (relative Einheiten): % (Prozent), em (Geviert)
  • Alternativ: x-small, small, medium, large, x-large
font-family
  • Historisch: Web-sichere Schriftarten
    • z.B. Verdana, Georgia, Times, Arial
    • Web-sichere Font-Stacks (Beispiele):
				font-family: Arial, Helvetica, sans-serif;
				font-family: Georgia, "Times New Roman", Times, serif;
				font-family: "Lucida Grande", Verdana, Arial, sans-serif;

Alternativen zu Web-sicheren Schriftarten: Web-Fonts

				@font-face {
				  font-family: "My-New-Font";
				  src: url("./my_font.ttf) format("truetype");
				}
				
				h1 { 
				  font-family: "My-New-Font", georgia, serif; 
				}

Web-Fonts: Probleme

  • Inkonsistente Unterstützung von Schriftformaten durch Browser
  • Lizenzierung von Schriften problematisch
  • Deshalb: Verwendung von Dienstleistern (Web Font Services)
