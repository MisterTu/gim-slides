Das TCP/IP-Referenzmodell

  1. Netzzugang – Ethernet, WiFi, ...
  2. Internet – IP
  3. Transport – TCP
  4. Anwendung – FTP, IMAP, HTTP, ...

1. Schicht: Netzzugang

  • Technische Basis für Datenübertragung
  • Punkt-zu-Punkt Datenübertragung
  • Beispiele: Ethernet, PPPoE, WiFi

2. Schicht: Internet

  • Internet Protocol (IP)
  • Beschreibt die Vermittlung und Wegewahl zwischen Computern
  • IP-Adresse – Eindeutiger Bezeichner für jeden Computer im Netz
  • Beispiel: 141.45.7.85

3. Schicht: Transport

  • Beschreibt den Datentransport zwischen Computern
  • Transmission Control Protocol (TCP)

4. Schicht: Anwendungen

  • Auf dieser Ebene arbeitet Anwendungssoftware
  • Beispiele:
    • IMAP – E-Mail
    • FTP – Dateitransfer
    • HTTP – Browser (WWW)

Bonus 1: Domain Name System (DNS)

  • Übersetzt Domain-Namen in IP-Adressen
  • „Telefonauskunft fürs Web”
  • htw-berlin.de in 141.45.7.85

Beispiel: nslookup htw-berlin.de

Bonus 2: Unified Resource Locator (URL)

  • Eindeutige Adresse eines Dokuments im Web
  • http://htw-berlin.de/Aktuelles.html
    • Verwendetes Protokoll – http
    • Domainname – htw-berlin
    • Top Level Domain – de
    • Pfad zur Datei – Aktuelles.html
