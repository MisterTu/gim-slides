Domain Name System (DNS)

  • Übersetzt Domain-Namen in IP-Adressen
  • „Telefonauskunft fürs Web”
  • htw-berlin.de in 141.45.7.85

Beispiel: nslookup htw-berlin.de

Unified Resource Locator (URL)

  • Eindeutige Adresse eines Dokuments im Web
  • http://htw-berlin.de/Aktuelles.html
    • Verwendetes Protokoll – http
    • Domainname – htw-berlin
    • Top Level Domain – de
    • Pfad zur Datei – Aktuelles.html

HTML: Doctype Declaration

Old School: XHTML1 
				<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
					"http://www.w3.org/TR/html4/strict.dtd">
New School: HTML5 
				<!doctype html>

HTML: Meta-Informationen

				<!doctype html>
				<html>
				  <head>
				    <title>Das ist der Titel der Seite </title>
				    <link rel="stylesheet" href="style.css" />
				  </head>
				(...)

HTML: Texte Strukturieren (Beispiele)

  • <h1> </h1> – Überschriften (h1 bis h6)
  • <p> </p> – Absätze
  • <em> </em> – Betonung (Emphasis)
  • <strong> </strong> – Starke Betonung

HTML: Sonstige Elemente (Beispiele)

  • <table> </table> – Tabellen
  • <ul> </ul> / <ol> </ol> – Listen
  • <a href="..."> </a> – Links (Anchor)
  • <img src="..." alt="..." /> – Bilder
  • <div> </div> – Abschnitt (nicht semantisch!)

HTML: Dokumente Strukturieren (Beispiele)

				<body>
				  <header>(...)</header>
				  <section>(...)</section>
				  <aside>(...)</aside>
				  <footer>(...)</footer>
				</body>

Darstellung vs. Inhalt

  • Structure – HTML
    • logische Strukturierung des Inhalts
  • Presentation – CSS
    • visuelle Darstellung
  • Behavior – JavaScript
    • Interaktion mit dem Nutzer
